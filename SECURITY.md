# Security Policy

## Supported Versions

A11y-Spy is currently in early development.

| Version | Supported |
| --- | --- |
| 0.x | Yes |
| Older experimental builds | No |

## Reporting a Vulnerability

Please do not report security vulnerabilities through public GitHub issues.

Instead, email:

```text
security@a11y-spy.dev
```

Include:

- Description of the vulnerability.
- Steps to reproduce.
- Affected version.
- Potential impact.
- Suggested fix, if available.

## Scope

Security reports may include:

- Malicious workspace file handling.
- Unsafe command execution.
- Dependency vulnerabilities.
- Supply-chain risks.
- Extension behavior that may expose local data.
- Unsafe auto-fix behavior.

## Out of Scope

The following are generally out of scope:

- Accessibility rule disagreements.
- False positives that do not create security risks.
- Bugs in third-party tools outside A11y-Spy.
- Issues caused by unsupported versions.

## Responsible Disclosure

Please give maintainers reasonable time to investigate and fix the issue before public disclosure.
